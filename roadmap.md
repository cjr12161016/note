## 🎯 牛客网 1400+ 分怎么拿？

牛客网的评分（类似 Elo 或天梯分）主要看算法题的解题能力和比赛表现。建议策略：

### 刷题计划
- **总量目标：** 150-200 题（含简单/中等/困难）
- **难度分布：** 简单 30% · 中等 50% · 困难 20%
- **频率：** 每天 2-3 题，周末可以集中打比赛

### 重点题型
| 题型 | 优先级 | 说明 |
|------|--------|------|
| 数组/字符串 | ⭐⭐⭐⭐⭐ | 最基础，必须熟练 |
| 链表操作 | ⭐⭐⭐⭐⭐ | 面试高频 |
| 二叉树/递归 | ⭐⭐⭐⭐⭐ | 核心思想 |
| 动态规划 | ⭐⭐⭐⭐ | 拉开差距的关键 |
| 排序/搜索 | ⭐⭐⭐⭐ | 基础功底 |
| 哈希表 | ⭐⭐⭐⭐ | 空间换时间 |
| 图论 | ⭐⭐⭐ | 经典题目即可 |
| 贪心 | ⭐⭐⭐ | 搭配 DP 练习 |

### 避坑指南
1. **不要追求难题** — 1400 分做好中等题就够了
2. **复盘 > 刷量** — 每道题想清楚为什么这么做
3. **及时回顾** — 每周复习前一周做错的题
4. **打周赛** — 牛客周赛是上分最快的方式

## 📚 推荐学习资源

### 编程基础
- **Python：** 廖雪峰 Python 教程（免费） + 《Python编程：从入门到实践》
- **C++：** 黑马程序员 C++ 教程（B站） + 《C++ Primer》选读

### 数据结构与算法
- **视频：** 尚硅谷 Java 数据结构（思路通用）
- **书籍：** 《大话数据结构》 或 《算法图解》（入门友好）
- **进阶：** 《剑指Offer》 + LeetCode 热题 100
- **网站：** 牛客网 · LeetCode · Codeforces

### 数学
- **线性代数：** 3Blue1Brown 线性代数本质（B站，必看！）
- **概率论：** 可汗学院 / 浙大概率论教材
- **离散数学：** 工作室考核常考，认真学课本

### 大数据 / 数据分析
- **SQL：** SQLZoo（在线练习） + LeetCode SQL 题库
- **Pandas：** 10 Minutes to Pandas 官方文档
- **实战：** Kaggle 入门比赛（Titanic、房价预测）
- **大数据：** 林子雨《大数据技术原理与应用》（入门推荐）
