# 02 - 数据结构与算法

> 牛客网 1400+ 的核心战场

---

## 刷题路线（按顺序！）

### 第一轮：基础数据结构（20天）

| 数据结构 | 推荐题数 | 关键点 |
|----------|---------|--------|
| 数组 | 10题 | 双指针、滑动窗口 |
| 链表 | 8题 | 反转、合并、环检测 |
| 栈/队列 | 8题 | 括号匹配、单调栈 |
| 哈希表 | 6题 | 两数之和、计数 |
| 二叉树 | 15题 | 遍历、构建、最近公共祖先 |

### 第二轮：核心算法（30天）

| 算法 | 推荐题数 | 关键点 |
|------|---------|--------|
| 排序 | 手写6种 | 快排、归并必须会手写 |
| 二分查找 | 8题 | 边界条件要搞清 |
| DFS/BFS | 10题 | 棋盘、岛屿、排列组合 |
| 动态规划 | 20题 | 背包、LIS、LCS、编辑距离 |
| 贪心 | 6题 | 区间、跳跃游戏 |
| 图论 | 8题 | 最短路、最小生成树、拓扑排序 |

### 第三轮：查漏补缺 + 综合（持续）

- 牛客周赛尽量参加
- 做错的题标记后一周内复习
- 整理题型模板
