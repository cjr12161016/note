## 📖 书籍推荐

### 入门级
| 书名 | 推荐理由 |
|------|---------|
| 《Python编程：从入门到实践》 | 最适合零基础的 Python 书 |
| 《算法图解》 | 可视化讲解算法，适合入门 |
| 《大话数据结构》 | 用故事讲数据结构，不枯燥 |

### 进阶级
| 书名 | 推荐理由 |
|------|---------|
| 《剑指Offer》 | 面试必刷，思路讲得好 |
| 《图解机器学习》 | 没有复杂公式，适合入门 ML |
| 《数据科学入门》 | 从零开始讲数据科学 |
