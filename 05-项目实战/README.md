# 05 - 项目实战

> 知识变成能力的地方，也是面试时最能打动人的部分

---

## 项目规划

### 项目一：数据分析入门（暑假前期）
- **内容：** 用 Pandas 分析一个公开数据集
- **数据集推荐：** 淘宝用户行为（阿里天池）、鸢尾花、房价数据
- **产出：** 数据清洗 → EDA → 可视化 → 结论报告
- **GitHub：** 用 Jupyter Notebook 展示分析过程

### 项目二：爬虫 + 分析（暑假后期）
- **内容：** 爬取某个网站数据并做分析
- **建议方向：** 知乎话题分析、豆瓣电影评分、天气数据
- **技术栈：** requests + BeautifulSoup / Scrapy + Pandas
- **产出：** 完整的爬取 → 存储 → 分析 → 可视化

### 项目三：简单机器学习（大二上）
- **内容：** 一个分类或回归预测任务
- **数据集：** Kaggle Titanic / 波士顿房价
- **技术栈：** Scikit-learn
- **产出：** 模型训练 → 评估 → 调参记录

---

## 项目标准

每个项目在 GitHub 上应该包含：

`
project-name/
├── README.md        # 项目介绍 + 运行方法
├── data/            # 数据（或数据来源说明）
├── notebooks/       # Jupyter Notebook 分析过程
├── src/             # 核心代码
└── reports/         # 分析报告（图表 + 结论）
`
