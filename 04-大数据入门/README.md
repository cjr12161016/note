---

## SQL（优先级：最高）

大数据工作室的面试大概率会考 SQL。

### 必须掌握的
| 知识点 | 说明 |
|--------|------|
| SELECT / WHERE / ORDER BY | 基础查询 |
| JOIN（INNER/LEFT/RIGHT） | 多表连接，必考 |
| GROUP BY + HAVING | 分组聚合 |
| 子查询 | 嵌套查询 |
| 窗口函数（ROW_NUMBER、RANK） | 面试加分项 |
| 建表、索引 | 基础DDL |

### 练习平台
- SQLZoo（免费在线，交互式）
- LeetCode SQL 题库（先做简单+中等）
- 牛客网 SQL 实战

## 数据分析（Python 生态）

### 工具链
`
Pandas（数据处理）
  → NumPy（数值计算）
    → Matplotlib / Seaborn（可视化）
      → Scikit-learn（机器学习基础）
`

### 入门项目推荐
1. **淘宝用户行为分析** — 用 Pandas 分析用户购买行为
2. **Titanic 生存预测** — Kaggle 经典入门
3. **天气数据分析** — 爬取历史天气数据 + 可视化

---

## 大数据工具概念（了解即可，大二深入学习）

| 工具 | 角色 | 一句话说明 |
|------|------|-----------|
| Hadoop HDFS | 分布式文件系统 | 存海量数据 |
| MapReduce | 分布式计算模型 | 分而治之 |
| Spark | 内存计算引擎 | 比 MapReduce 快很多 |
| Hive | 数据仓库 | 用 SQL 查 Hadoop |
| Flink | 实时流处理 | 处理实时数据 |

**推荐入门书：** 林子雨《大数据技术原理与应用》（免费电子版）
