# 01 - 编程基础

> Python 和 C/C++ 是进入大数据领域的左膀右臂

---

## 🐍 Python（重点！）

大数据工作室最常用的语言就是 Python。**优先级最高！**

### 需要掌握的内容

| 模块 | 要求 | 备注 |
|------|------|------|
| 基础语法 | 变量、循环、条件、函数、类 | 必须闭眼写 |
| 文件操作 | 读写 txt/csv/json | 数据预处理的基础 |
| NumPy | 数组操作、矩阵运算 | 数据分析核心库 |
| Pandas | DataFrame、GroupBy、merge | 必会！面试常考 |
| Matplotlib | 折线图、柱状图、散点图 | 数据可视化基础 |

### 推荐学习路径
1. 廖雪峰 Python 教程 → 2 周过一遍
2. 菜鸟教程 Python 100 例 → 每天练几道
3. 动手做一个小项目：爬取百度热搜 + 简单分析

---

## ⚡ C/C++

牛客网刷题的主力语言之一。**优先级次之**，但建议会用。

### 需要掌握的内容
| 模块 | 要求 |
|------|------|
| 基本语法 | 输入输出、数组、字符串、指针 |
| STL 容器 | vector、stack、queue、map、set |
| STL 算法 | sort、lower_bound、next_permutation |
| 内存管理 | new/delete、深拷贝浅拷贝概念 |

### 建议
- 不需要学太深，能看懂、能写简单题就行
- 刷题时 Python 和 C++ 选一个为主（推荐 Python）
