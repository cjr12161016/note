## 大数据工作室一般考什么？

根据川农大工作室的经验，一般考核包含：

### 笔试 / 机试
| 内容 | 比例 | 说明 |
|------|------|------|
| 数据结构与算法 | 40% | 牛客网风格的算法题 |
| SQL | 20% | 查询题 |
| 数学基础 | 20% | 概率统计、线性代数 |
| 逻辑题 | 20% | 智力题 + 开放问题 |

### 面试
| 环节 | 内容 |
|------|------|
| 自我介绍 | 1-2 分钟，突出学习热情和基础 |
| 项目问答 | 做了什么、遇到什么问题、怎么解决的 |
| 基础知识 | 编程语言、数据库、数据结构 |
| 开放问题 | 为什么想来工作室、未来规划 |

---

## 面试准备清单

### 技术面
- [ ] 手撕代码：链表反转、二叉树遍历、快排
- [ ] SQL：多表 JOIN + GROUP BY
- [ ] Python：列表推导式、装饰器概念、Pandas 基础
- [ ] 项目：能流畅讲出项目逻辑和难点

### 软技能
- [ ] 准备一个清晰的自我介绍
- [ ] 想清楚为什么转专业
- [ ] 想清楚为什么想进大数据工作室
- [ ] 了解工作室最近做过的项目

## 加分项（现在就可以做）

1. **逛 GitHub** — 关注大数据相关的项目，学习 README 怎么写
2. **关注牛客网面经** — 看川农同学分享的面试经验
3. **写博客** — 把自己的学习过程写成文章（掘金、知乎、CSDN都行）
4. **参与开源** — 哪怕只是提一个 issue 或改一个 typo
