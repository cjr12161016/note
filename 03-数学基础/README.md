# 03 - 数学基础

> 信息与计算科学的灵魂，大数据分析的底层支撑

---

## 为什么要学？

大数据工作室考核和日常工作中，数学无处不在：
- **线性代数** → 数据降维（PCA）、推荐系统（SVD）
- **概率统计** → A/B测试、贝叶斯分类、置信区间
- **离散数学** → 图论在社交网络分析中的应用

---

## 笔记方法

数学笔记建议按「**概念 → 公式 → 代码实现 → 应用场景**」四步走：
- 先理解概念（看 3Blue1Brown 视频）
- 记住关键公式
- 用 Python 写一段代码验证（比如用 NumPy 验证矩阵乘法）
- 思考这个知识点在大数据领域用在哪里
